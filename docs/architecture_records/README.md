# Architecture Decision Records (ADRs)

This folder tracks key architectural decisions made in the project. Each ADR documents context, decision, and consequences.

- ADR-001: Use Cloudinary for production image storage
- ADR-002: Use Google OAuth2 for authentication
- ADR-003: Use Twilio Verify for phone verification (SMS)
- ADR-004: Image upload strategy (service object) and local vs cloud behavior

Conventions:
- New ADRs use the next sequential number
- Keep ADRs short and actionable; prefer links to code or docs for details
