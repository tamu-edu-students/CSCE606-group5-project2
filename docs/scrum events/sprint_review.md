
## **Sprint Review**

During the Sprint review meetings, we demonstrated all completed features and discussed progress against the sprint backlog.
Major accomplishments showcased:

* **Authentication & Profiles UIs:** Login/logout, Google OAuth Integration
* **Item management:** Add Item Listing, Edit Listing, View all listings, Upload Images using Cloudinary
* **Messaging and Requests:** CRUD functionality for messages between users, Requests on items 
* **Profile Verification, Virtual Accounts:** Profile Verfication using Twilio Extension, Seeding data to help test application
* **Sorting, Filtering, Search Items:** Item search, Item listing sorting by price/categort/name, and Item Filtering.
* **Profile Ratings:** Buyers can rate sellers, Ratings displayed on Seller UI webpage
* **Testing and CI/CD:** RSpec and Cucumber tests added for user flows; app deployed successfully on Heroku.

All the **Sprint goals were met**, with all key user stories completed and merged into `dev` after review.
